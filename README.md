# Wexo - WhatsApp Business SaaS Platform

Production-ready WhatsApp Business API SaaS with AI, CRM, Campaigns, and Billing.

## Features

- ✅ WhatsApp Business API Integration
- ✅ Real-time Inbox with Socket.IO
- ✅ Visual Flow Builder
- ✅ Bulk Campaigns with Scheduling
- ✅ Contact CRM with Tags & Segments
- ✅ AI Auto-Replies (Gemini 2.0)
- ✅ Order Management from Catalog
- ✅ Analytics Dashboard
- ✅ Stripe Subscriptions (3 Plans)
- ✅ RBAC (Owner, Admin, Agent, Viewer)
- ✅ BullMQ Background Jobs
- ✅ Redis Caching

## Tech Stack

- **Runtime**: Node.js 20 + TypeScript
- **Framework**: Express.js
- **Database**: PostgreSQL + Prisma ORM
- **Cache/Queue**: Redis + BullMQ
- **Real-time**: Socket.IO
- **AI**: Google Gemini 2.0 Flash
- **Payments**: Stripe
- **WhatsApp**: Meta Graph API v21.0
